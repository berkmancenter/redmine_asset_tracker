Redmine Asset Tracker

The redmine asset tracker is a project created by the Berkman Center at Harvard University, Coded by Emmanuel Pastor and sponsored by Google through the Google Summer of Code program.

It is a Redmine plugin with absolutely zero third party dependencies other than Ruby on Rails and Redmine itself.

This project is Open Source, and can be checked out from Github.
